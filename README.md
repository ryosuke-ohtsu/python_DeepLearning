# 顔認識システム

カメラで撮影した顔画像を学習して、キャラクター・人物を認識するシステムです。

## 📋 セットアップ

### 必要なライブラリ
```bash
pip install opencv-python opencv-contrib-python pillow numpy
```

## 🚀 使い方

### 統合メニュー版（推奨）

```bash
python main.py
```

メニュー画面で以下を選択できます：
- **1. 顔画像の撮影・登録** → カメラで複数人の顔を撮影・保存
- **2. モデルの学習** → 保存した顔画像を学習
- **3. 顔の認識** → 学習したモデルで顔を認識
- **4. 登録済み人物の確認** → 登録済みの人数・枚数を表示
- **5. 終了** → プログラムを終了

### 個別スクリプト版

お好みに応じて個別のスクリプトも利用可能：

```bash
# 1. 顔画像をカメラで撮影
python face_collect.py

# 2. 撮影した画像を学習
python train_faces.py

# 3. 学習したモデルで認識
python recognize_faces.py
```

## 💡 操作ガイド

### 顔画像の撮影
1. メニューで「1. 顔画像の撮影・登録」を選択
2. 人物の名前を入力
3. カメラに映った顔を正面に向ける
4. **S キー** を押すと顔画像が保存されます（10枚～20枚推奨）
5. **Q キー** で次の人物に
6. 空欄で入力すると撮影を終了

### モデルの学習
1. メニューで「2. モデルの学習」を選択
2. 自動で `dataset/` フォルダをスキャン
3. 学習が完了すると `trainer/` に モデルが保存されます

### 顔の認識
1. メニューで「3. 顔の認識」を選択
2. カメラに映った顔が自動認識されます
3. **Q キー** で終了

## ⚙️ パラメータ調整

`main.py` 内のパラメータを調整することで精度を改善できます：

### 顔検出関連
- `scaleFactor=1.2` → 顔検出の粗さ。小さいほど正確ですが遅い
- `minNeighbors=5` → 小さいほど多く検出、大きいほど厳選
- `minSize=(100, 100)` → 最小顔サイズ

### 認識精度関連
- `confidence < 60` → 一致判定の閾値。小さいほど厳密
- `confidence < 100` → 候補表示の閾値

## 📊 精度向上のコツ

1. **各人物5～20枚の顔画像を保存**
   - 正面、左向き、右向きなど角度を変える
   - 照明を変えて撮影

2. **背景を統一**
   - なるべく同じ背景で撮影

3. **顔サイズを統一**
   - カメラまでの距離をほぼ一定にする

4. **定期的に再学習**
   - 新しい人物を追加したら必ず再学習

## 🛠️ トラブルシューティング

**Q. カメラが認識されない**
- A. USB カメラの場合、別途ドライバをインストール
- 内蔵カメラの場合、別のアプリでテスト

**Q. 顔が正しく認識されない**
- A. 画像枚数を増やす（最低10枚）
- 再度撮影して学習し直す

**Q. エラーが出る**
- A. 必要なライブラリをインストール:
```bash
pip install --upgrade opencv-python opencv-contrib-python pillow numpy
```
